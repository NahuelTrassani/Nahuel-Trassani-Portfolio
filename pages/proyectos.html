Proyectos

E-Commerce React

Aplicación de comercio electrónico desarrollada con React.js, Redux para manejo de estado y Node.js/Express para el backend. Incluye sistema de autenticación, carrito de compras, procesamiento de pagos con Stripe y panel de administración. Base de datos MongoDB.

React Node.js MongoDB Redux

Task Manager App

Aplicación web para gestión de tareas y proyectos con funcionalidades de colaboración en tiempo real. Desarrollada con Vue.js en el frontend, Laravel en el backend y Socket.io para comunicación en tiempo real. Incluye notificaciones push y sistema de roles.

Vue.js Laravel MySQL Socket.io

Dashboard Analytics

Panel de control interactivo para visualización de datos y métricas de negocio. Implementado con Angular y D3.js para gráficos dinámicos, conectado a una API REST desarrollada en Python Flask. Incluye filtros avanzados y exportación de reportes en PDF.

Angular Python Flask D3.js

App Móvil Fitness

Aplicación móvil multiplataforma para seguimiento de rutinas de ejercicio y nutrición. Desarrollada con React Native, incluye integración con sensores del dispositivo, sincronización con wearables, y gamificación con sistema de logros y desafíos.

React Native Firebase REST API Redux

CMS Blog Personal

Sistema de gestión de contenidos personalizado para blog con editor de texto enriquecido, sistema de comentarios, categorización de posts y SEO optimizado. Desarrollado con PHP puro, MySQL y implementación de cache para mejor rendimiento.

PHP MySQL JavaScript Bootstrap
